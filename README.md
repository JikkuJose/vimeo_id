# vimeo_id

Gem to extract video id from various Vimeo urls

## Installation

Add this line to your application's Gemfile:

    gem 'vimeo_id'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vimeo_id

## Usage

    VimeoId.from('')
